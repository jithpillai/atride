# AtRide brand assets

These are the approved working raster assets for the AtRide product. The logo
geometry should be redrawn as SVG after the visual system is finalized.

## Logo assets

| Asset | Intended use |
| --- | --- |
| `domain-lockup.png` | Original transparent symbol + `atride.in` domain lockup |
| `domain-lockup-light.png` | Domain lockup retained for light backgrounds |
| `domain-lockup-dark.png` | White/orange domain lockup used on dark backgrounds |
| `primary-logo.png` | Transparent symbol + `@Ride` for light backgrounds |
| `primary-logo-dark.png` | Transparent light/orange variant for dark backgrounds |
| `primary-logo-monochrome.png` | Transparent single-colour charcoal logo |
| `primary-logo-monochrome-white.png` | Transparent single-colour white reverse logo |
| `symbol-only.png` | Transparent full symbol without a wordmark |

## App and browser icons

| Asset | Intended use |
| --- | --- |
| `app-icon-maskable-master.png` | Full-size, full-bleed icon source |
| `app-icon-512.png` | PWA 512 × 512 maskable icon |
| `app-icon-192.png` | PWA 192 × 192 icon |
| `apple-touch-icon-180.png` | Apple touch icon |
| `favicon-48.png` | Large browser favicon |
| `favicon-32.png` | Standard browser favicon |
| `favicon-16.png` | Small browser favicon |
| `favicon.ico` | Legacy browser favicon |

The `*-chroma.png` files are intermediate green-screen sources retained only
for regenerating the transparent PNGs. They must not be used by the web app.
